# DPGKモード 改修タスク — 実行ガイド（sonnet 4.6 向け）

このディレクトリは、`やること.txt` の DPGKモード改修5項目を **sonnet 4.6 に1タスクずつ実行させる**ための指示書セットです。

## 使い方（重要：コンテキスト節約）

各セッションで読むのは **2ファイルだけ** にしてください。

1. `01_context.md` … アーキテクチャ・コードマップ・共通ルール（毎回これだけは読む）
2. `task-N-*.md` … 今回実行する1タスク

タスクファイルは自己完結しています。複数タスクを1セッションに詰め込まず、**1タスク = 1セッション**を推奨します（窓 約20万トークン。コードを丸読みすると圧迫するため、`01_context.md` に書かれた該当箇所だけを `Read` の `offset`/`limit` で部分的に開くこと）。

## タスク一覧と推奨実行順

依存とコンフリクトを避けるための順番です。

| 順 | ファイル | 内容 | 主に触る所 |
| --- | --- | --- | --- |
| 1 | `task-5-exit-button.md` | ✖ → 「退出」ボタン化 | DpgkApp.jsx / app.css（小） |
| 2 | `task-1-controls-under-video.md` | 操作ボタンを動画の真下へ | DpgkApp.jsx 構造 / app.css |
| 3 | `task-3-volume-control.md` | 音量調整UIを追加 | useTwitchPlayer.js / DpgkApp.jsx / app.css |
| 4 | `task-4-reload-mute-persist.md` | リロードでミュート固定される問題 | useTwitchPlayer.js |
| 5 | `task-2-twitch-overlay-hide.md` | Twitch純正オーバーレイ（ミュート表示）が消えない | useTwitchPlayer.js / DpgkApp.jsx |

- タスク2・3・4はプレイヤー（`useTwitchPlayer.js`）に関連し、互いに絡みます。**4→2 の順**だと「ミュート解除で純正の消えないオーバーレイ自体が出なくなる」可能性があり、2の作業が軽くなります。
- タスク1（レイアウト構造変更）はタスク3（音量UIの置き場所）に影響するため、**1→3 の順**で。

## 厳守ルール

- **絶対に `git commit` / `git push` しない。** ユーザーが明示的に「コミットして」と言うまで実行禁止（このリポジトリのグローバルルール）。変更を終えたら内容を説明して止まる。
- ブランチは切らず `main` 上で作業してよい（コミットはしない）。
- 既存コードのスタイル（日本語コメント・命名・2スペースインデント）に合わせる。
- 各タスクの「完了条件」と「検証手順」を満たしてから報告する。
